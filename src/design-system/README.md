# Design System (Reusable UI)

This folder contains reusable, branded, Tailwind-based UI components for future pages.

## Notes

- Components are future-proofed for dark mode via `dark:` utility classes.
- Some components support loading states (e.g., `Button`, `LoadingSkeleton`, `StatCard`).
- No existing app pages/auth/routing/Supabase integration were modified.
